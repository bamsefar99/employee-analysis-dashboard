
# Employee Analysis Dashboard

This project provides tools and scripts to analyze employee data, focusing on **age distribution** and **blue-collar vs. white-collar segmentation**. The dataset includes employee names, birth dates, start dates, and job roles.

## Objectives
- **Age Distribution Analysis**  
  Visualize how employees are distributed across different age groups.

- **Blue-Collar vs. White-Collar Segmentation**  
  Classify roles into:
  - **Blue-Collar**: Warehouse (Lager)
  - **White-Collar**: Administration, Sales, Customer Service

## Features
- Generate charts for age distribution.
- Calculate percentage split between blue-collar and white-collar employees.
- Export results as visual reports.
